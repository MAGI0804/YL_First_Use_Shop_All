# 第七周基础修复更新记录（2026-06-03）

分支：`develop`

## 本次范围

- 第 7 周评价体系和数据分析基础修复。

## 更新记录

### 第一批：Web 评价管理和报表真实接口

- Web 新增评价管理菜单和路由：`/reviews`。
- Web 新增评价管理页，接入 `POST /review/query_backend` 查询评价列表。
- Web 评价管理支持审核通过、拒绝、隐藏和商家回复，分别接入 `POST /review/audit`、`POST /review/reply`。
- Web 评价管理接入 `POST /review/statistics` 展示评价总数、平均评分、好评率和待审核数。
- Web 报表页移除静态占位数据，接入 `POST /analytics/sales_summary`、`POST /analytics/user_summary`、`POST /analytics/product_summary`。
- Web 报表页展示销售额、已支付订单、优惠金额、退款金额、客单价、日销售明细、热销 SKU、热销款号、用户偏好、库存周转和评价指标。
- Web 报表页接入 `POST /analytics/export`，支持导出结构化 JSON 报表。

### 第二批：小程序评价入口和商品详情评价展示

- 小程序订单详情在已签收商品上展示评价入口。
- 小程序订单详情通过 `POST /order/query_sub_order_data` 补齐评价所需的子订单号。
- 小程序新增评价提交页：`pages/my/order/review/index`。
- 小程序评价提交接入 `POST /review/create`，提交评分、评价内容和评价标签。
- 小程序商品详情接入 `POST /review/statistics` 展示评价总数、平均评分和好评率。
- 小程序商品详情接入 `POST /review/query_by_product`，仅展示审核通过的商品评价和商家回复。
